Sobre mí

Tecnólogo en Desarrollo de Software con experiencia llevando proyectos propios desde cero hasta producción. He construido un e-commerce completo con pasarela de pagos, una app Android nativa y múltiples proyectos colaborativos, siempre aplicando arquitectura limpia y metodologías ágiles.

Mi enfoque actual combina desarrollo full stack con inteligencia artificial aplicada — uso modelos de IA locales y asistentes con Spec-Driven Development (SDD) para planificar, revisar y automatizar tareas del flujo de trabajo. No es una moda, es una herramienta más en mi cinturón.

Proyectos destacados

Full Stack

Victorem 18K

E-commerce completo para venta de joyas artesanales en oro laminado 18K. Integra frontend, backend, base de datos y pasarela de pagos real.

  • Next.js + TypeScript con sistema de diseño propio en CSS Custom Properties
  • Backend REST Node.js/Express con autenticación JWT y refresh token
  • PostgreSQL en Neon con inicialización via scripts
  • Pasarela Wompi: tarjeta, Nequi, PSE, Bancolombia
  • CI/CD con GitHub Actions + Vitest
  • Flujo SDD con agentes de IA para planificar y revisar
Código fuente
Android

MovieApp

App Android nativa para explorar películas populares, consultar detalles y guardar favoritos. Proyecto final de la materia Dispositivos Móviles.

  • Kotlin + Jetpack Compose con arquitectura MVVM
  • Consumo de API REST (TMDB) con Retrofit y Corrutinas
  • Favoritos persistidos localmente con Room (SQLite)
  • Modo oscuro/claro, búsqueda y navegación entre pantallas
  • StateFlow para reactividad y estado de UI
